Frequently asked questions

When do I get my first commission?
14 days after the client pays (end of trial). Payment via PayID every 15 days (day 15 and day 30).
If my client cancels, do I lose the commission?
No, if the commission was already released (commission_ready), it's yours. Later cancellation doesn't claw it back. It only doesn't count if the client cancels within the 14-day window.
Can I change my code later?
No — once created, the code is fixed to preserve historical tracking. Choose a memorable one (your brand, your initials).
Is there a fee to be an affiliate?
Zero. The program is 100% free. You also get lifetime PRO app access to use with clients.
Client used my link but not my code — do I earn?
No — attribution is by the code applied in app setup. Always instruct your client: "after downloading, in setup, paste the code YOURCODE".
Is there a referral limit?
No. The more, the better. Volume bonuses scale up to 20+/month.
Can I refer outside Australia?
The app works best for AU users (local tax). Referrals outside AU don't convert well — focus on AU.
